Το πρώτο πράγμα που έκανα ήταν να φτίαξω ένα απλό starter template στο Adobe Dreamweaver και μετά να οργανώσω τον κώδικα για να μπορώ να τα βλέπω όλα πιο εύκολα. Μετά, έκανα stylize τα διάφορα element και έβαλα ένα controllable image slider που βρήκα εδώ: https://getbootstrap.com/docs/4.6/components/carousel/#slides-only και το έκανα stylize. Μετά, έφτιαξα τα ABOUT, EVENTS, DONATE, & CONTACT pages με την βοήθεια διαφόρον templates που έκανα modify και με τις εικόνες που σας δίνω παρακάτω




Coding languages/technologies used: HTML5, CSS3, Bootstrap 4, Bootstrap's JavaScript & CSS plugins, Popper
Programs used: Adobe Dreamweaver for coding the page & Adobe Photoshop for the sitemap
Templates used: Several of Dreamweaver's templates were used but slightly modified. A modified version of Humanity's Bootstrap template's donation page was used for my page's donate page & a modified version of a template which was found here: https://startbootstrap.com/template/heroic-features was used for the events page
Icons used: https://icons.getbootstrap.com/
